# Campus Companion App

## Overview

Campus Companion is a simple mobile app made with React Native and Expo.
It helps new students quickly find useful campus information.

Main features:

- Home screen with quick buttons
- Important Contacts list and contact details page
- Weekly class schedule view
- Notice Board for campus updates
- Light and dark theme toggle

## Install and Run

You can use npm or yarn.

1. Install dependencies

- npm: `npm install`
- yarn: `yarn install`

2. Start the app

- npm: `npm start`
- yarn: `yarn start`

3. Run on device/emulator

- Android (npm): `npm run android`
- iOS (npm): `npm run ios`
- Web (npm): `npm run web`
- Android (yarn): `yarn android`
- iOS (yarn): `yarn ios`
- Web (yarn): `yarn web`

## Known Issues / Limitations

- Data is static (no backend/database yet).
- Theme choice is not saved after app restart.
- Notice Board and Schedule are demo data only.
